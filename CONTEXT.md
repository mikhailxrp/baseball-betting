# Baseball Betting System — контекст проекта

## Что это

Рекомендательная система ставок на бейсбол MLB.
Агент анализирует матчи и рекомендует ставки на тоталы.

## Как работает

1. Получаем расписание матчей на день с MLB Stats API
2. Агент анализирует питчеров, составы, статистику команд
3. Система выдаёт рекомендацию: матч / тип ставки / линия / уверенность / причины
4. Пользователь делает ставку у букмекера вручную
5. Результат вводится в систему вручную
6. После накопления статистики включается финансовая система

## Тип ставок

- total_over — общий тотал больше
- total_under — общий тотал меньше
- ind_total_over — индивидуальный тотал команды больше

## Источники данных

- MLB Stats API (бесплатно, без ключа):
  https://statsapi.mlb.com/api/v1
- OpenWeather API (погода)

## База данных (Supabase / PostgreSQL)

Таблицы в порядке зависимостей:

1. teams — справочник команд
2. games — матчи
3. team_stats — статистика команд по сезонам
4. bets — ставки
5. bank — история банка

## Ключевая логика анализа

Тотал высокий когда:

- Питчер слабый (высокий ERA, FIP, WHIP)
- Команда атакующая (высокий R/G, OPS, wRC+)
- Высокий LOB в предыдущем матче серии (не реализовали преимущество)
- Мотивация команды высокая (борьба за плей-офф)

## Финансовая система

Включается когда команда набрала 10+ ставок и показывает стабильность.
Расчёт суммы догонялки:
следующая ставка = (проигрыш + желаемая прибыль) / (коэф - 1)
